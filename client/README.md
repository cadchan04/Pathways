# Client (React)

## Tech Stack
- React (Vite)
- React Router
- Axios

## Setup
```bash
cd client
npm install
npm run dev
```

# Backend (Node.js)

## Tech Stack
- Node.js
- Express
- CORS

## Setup
1. Download and install Node from https://nodejs.org/en/download
2. Open separate terminal from the one running the client
```bash
cd server
npm install
node app.js
```
